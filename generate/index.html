Генератор тренажёров

Заполните тему и список слов. Заглавные подряд идущие группы превращаются в поля ввода, а английская X означает пустое место.

Формат: группы заглавных букв = пропуск, английская X = пустой пропуск.
Пример строк: вОрОнА · кОрОвА · гектарОВ · бахилX
